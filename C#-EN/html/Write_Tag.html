Write Tag Memory

WriteTagMemWords Method

Write data into memory area of a bank
 
Method Signature
  
public abstract void WriteTagMemWords(ModuleTech.TagFilter target, ModuleTech.Gen2.MemBank bank, int address, ushort[] data)
  
Parameters
Parameter Description
target Tag filter, set as null if it is not necessary.
bank Bank to write
address The starting address, in blocks, in the bank.
data Data to write
  
Example
Write two blocks of data "0x44445555" from the third block into USER bank.
  
string datastr = "0x44445555";
ushort[] writedata = new ushort[2];
for (int a = 0; a < writedata.Length; ++a)
	writedata[a] = ushort.Parse(datastr.Substring(a * 4, 4), System.Globalization.NumberStyles.AllowHexSpecifier);
rdr.WriteTagMemWords(null, MemBank.USER, 3, writedata);
